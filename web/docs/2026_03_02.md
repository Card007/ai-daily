### 6. Shell 变量 ~-

[Shell variable ~-](https://www.johndcook.com/blog/2026/03/01/tilde-dash/) — **johndcook.com** · 03-01 18:15 · ⭐ 16/30

> bash 里存在一个不太常见但很实用的路径快捷方式：~-，它等价于 shell 变量 $OLDPWD（上一次工作目录）。很多人熟悉用 cd - 在当前目录与上一个目录之间来回切换，但 ~- 允许把“上一个目录”当作路径片段直接参与命令。这样可以写出类似 ls ~-、cat ~-/file、cp a ~-/backup 之类的操作，而不需要先 cd - 再执行命令。要点是：~- 让“上一个目录”不仅能跳转，还能在任意需要路径的地方复用。作者的结论是这是一个提升命令行效率的小技巧，特别适合频繁在两个目录间切换并进行文件操作的场景。

🏷️ bash, shell, OLDPWD, productivity
