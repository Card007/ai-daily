### 20. Three ways to differentiate ReLU

[Three ways to differentiate ReLU](https://www.johndcook.com/blog/2026/04/30/derivative-of-relu/) — **johndcook.com** · 04-30 14:51 · ⭐ 21/30

> ReLU, defined as max(0, x), is not classically differentiable at x = 0, which forces you to choose a notion of “generalized derivative.” One option is to use the ordinary derivative where it exists, giving 0 for x < 0 and 1 for x > 0, and treat the single non-differentiable point as negligible for many purposes. A second option is the convex-analysis subgradient, where the derivative at 0 becomes a set of valid slopes (typically any value in [0, 1]), matching how optimization methods justify “pick something” at kinks. A third option is the distribution/weak-derivative viewpoint, which formalizes the derivative via integration against test functions and yields a principled object corresponding to the step behavior of the ReLU slope. The takeaway is that “the derivative of ReLU at 0” is not a single number until you specify which derivative concept you are using, and different contexts (optimization vs. analysis) naturally prefer different choices.
